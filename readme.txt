Improvements and Fixes for WordPress

== Description ==

A collection of useful "[function methods](https://ifwp.vidsoe.com/#function-mehotds "Function methods")" for your WordPress theme's functions.php.

[Usage](https://ifwp.vidsoe.com "Usage")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ifwp` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `ifwp` function in your WordPress theme's functions.php.
